# Remotes
To list remote repos:
```
> git remote -v
```
To add a new remote:
```
git remote add <shortname> <url>
```
To fetch a remote:
```
git fetch <shortname>
```
To push to a remote:
```
git push <shortname> <branch>
```
To inspect a remote:
```
git remote show <shortname>
```
To rename a remote:
```
git remote rename <oldshortname> <newshortname>
```
To remove a remote:
```
git remote rm <shortname>
```
